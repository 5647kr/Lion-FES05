## 궁금한 점
```
<style>
html {
  font-size: 10px;
}
.hello2 {
  font-size: 1rem;
}
  .strong2 {
  font-size: 0.5rem;
}
</style>

  <p class="hello2">p tag2 rem
    <strong class="strong2">strong tag2 rem</strong>
  </p>
```
